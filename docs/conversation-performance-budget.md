# Conversation Performance Budget

## 목표 기준
- 목표 프레임: **60 FPS** (저사양 구간 최소 45 FPS 유지)
- 1프레임 CPU 예산: **16.6ms 이하**
- 장면 전환 피크 메모리: **+35MB 이내**
- 안정 구간 JS Heap 목표: **120MB 이하**

## 품질 정책
- `QualityProfile` 프리셋(`low/medium/high`)으로 텍스트 타이핑, 파티클, 블러, 동시 Tween 수를 제한한다.
- 긴 대사 구간은 텍스트 오브젝트 풀을 재사용해 GC 급증을 완화한다.
- 백그라운드/비활성 전환 후 타이머는 wall-clock 보정 스케줄러로 재동기화한다.

## 체크 항목
1. 카드 장면 자동 진행과 종장 힌트 타이밍이 백그라운드 복귀 직후에도 의도 시간대로 유지되는지 확인.
2. `spawnTextureSparkles` 호출 시 프리셋별 파티클 최대치가 적용되는지 확인.
3. 저사양 환경에서 블러 연출이 비활성화되는지 확인.
4. 긴 리딩 반복 실행 시 텍스트 생성/삭제가 아닌 풀 재사용으로 힙 변동폭이 감소하는지 확인.
